# SeansLifeArchive_Images_Idle_Miner_Tycoon
The module for my life story project that contains my Idle Miner Tycoon images.

***

## About

See above. This repository hosts all my weekly Idle Miner Tycoon session pictures. I currently play the [Android version](https://play.google.com/store/apps/details?id=com.fluffyfairygames.idleminertycoon&hl=en_US&gl=US).

These daily pictures are to be used for progress monitoring, but can also be used as stock images.

I started playing Idle Miner Tycoon back in 2018. I played it on and off a few times.

I originally started playing from 2018 to 2019, then I had a long break before I began playing it again in 2020.
